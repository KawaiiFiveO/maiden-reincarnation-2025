A. The Debutante Dilemma

Time limit per test: 2 seconds
Memory limit per test: 1024 megabytes

In the royal capital of Eorka, the grand Spring Ball is being held at the Imperial Palace. This ball marks the debut of noble ladies into high society. Every eligible noble family sends exactly one daughter (unless they're scandalous enough not to).

Penelope, the villainess (but totally misunderstood), has received a guest list with the names of the debutantes. However, she suspects that some commoners might have snuck in by faking noble names. Unacceptable!

In Eorka, noble names are always unique — no two debutantes should share the same name. If there are any duplicates, the palace guards will start investigating.

Can you help Penelope determine if the list is valid (all names are unique) or scandalous (there are duplicates)?

Input

The first line contains a single integer \( n \) \((1 ≤ n ≤ 100)\), the number of debutantes.

The next \( n \) lines each contain a single string \( s_i \) — the name of the \( i \)-th debutante. Each name is a string of lowercase English letters, length between 1 and 20.

Output

Print Valid if all names are unique. Otherwise, print Scandalous.

Example

Input

5
evelyne
beatrice
yuri
beatrice
sophia

Output

Scandalous

Input

6
isabelle
victoria
noelle
shuli
katalina
penelope

Output

Valid

Explanation

In the first example, Beatrice is listed twice — how dare she show up twice?! So the list is scandalous. In the second example, everyone's name is unique, so the list is valid.

B. Time Travel Paranoia

Time limit per test: 2 seconds
Memory limit per test: 1024 megabytes

Rosalite discovers that she's stuck in a time loop. Every time her brother, Asterion, dies, she gets sent back to the same point in time!

She keeps a journal to track how many things she does each loop. The journal records her task counts for every loop.

After \( n \) loops, she wants to know: On how many loops did she do strictly more tasks than the previous loop?

Input

The first line contains a single integer \( n \) \((2 ≤ n ≤ 1000)\).

The second line contains \( n \) integers \( a_1, a_2, ..., a_n \), where \( a_i \) is the number of tasks she completed on the \( i \)-th loop. \((0 ≤ a_i ≤ 10^4)\)

Output

Print a single integer: the number of loops where \( a_i > a_{i-1} \).

Example

Input

6
3 5 5 6 2 4

Output

3

Explanation

Loop 2 > Loop 1 → Yes
Loop 3 = Loop 2 → No
Loop 4 > Loop 3 → Yes
Loop 5 < Loop 4 → No
Loop 6 > Loop 5 → Yes

Total: 3 loops where she improved.

C. Kidnapping the Male Lead

Time limit per test: 2 seconds
Memory limit per test: 1024 megabytes

Vivian got into a carriage accident and suddenly remembers everything: her future death, the demon's rise, and worst of all — the only clue about the male lead:

He was the most handsome man in the empire.

That's it. No name. No birthmark. No tragic backstory. Just "handsome."

Unfortunately for her, this world is absolutely infested with hot men.

Determined to fix the plot, Vivian begins kidnapping the most handsome men she encounters, one by one, hoping to find the real male lead.

But now, over 30 kidnappings in, she's desperate. She wants to know if any of them might still be the guy she's looking for.

Her logic is simple:

The real male lead is the only man who has the highest possible beauty score.
She kidnapped a bunch of guys and recorded their scores.
Can you help her identify if there's only one possible male lead?

Input

The first line contains an integer \( n \) \( (2 ≤ n ≤ 10^5) \), the number of kidnapped men.

The second line contains \( n \) integers \( a_1, a_2, ..., a_n \) \( (1 ≤ a_i ≤ 100) \), where \( a_i \) is the beauty score of the \( i \)-th kidnapped man.

Output

If the highest beauty score appears only once, print Found him! — he's probably the male lead.

If multiple men share the top score, print Who knows? — she still has work to do.

Example

Input

5
91 85 97 88 97

Output

Who knows?

Input

3
89 92 85

Output

Found him!

D. Rondo of the Heroine and Reasoning

Time limit per test: 2 seconds
Memory limit per test: 1024 megabytes

On the quiet island of Rokkenjima, a heinous crime has been committed. There are \( n \) suspects numbered from 1 to \( n \). Each suspect gives an alibi by naming one person who they believe is innocent. Every suspect except the killer is named as innocent by at least one person.

The heroine, Erika, arrives on the scene. Erika's task is to find the killer — the one who is never vouched for by anyone.

Input

The first line contains an integer \( n \) \( (2 ≤ n ≤ 100)\) — the number of suspects.

The second line contains \(n\) integers \(a_1, a_2, …, a_n \) \((1 ≤ a_i ≤ n)\), where \(a_i\) represents the suspect that the \(i\)-th suspect claims is innocent.

Output

Print a single integer — the number of the killer (i.e., the suspect who is never declared innocent by any other suspect). It is guaranteed that only one killer exists.

Example

Input

4
2 3 1 1

Output

4

Explanation

Suspects 1, 2, and 3 are declared innocent by someone. Only suspect 4 is not declared innocent by any suspect. Therefore, suspect 4 is the killer.

E. The Great Estate Developer Knows

Time limit per test: 2 seconds
Memory limit per test: 1024 megabytes

Lloyd has been commissioned to solve a water crisis in Frontera, a bustling town on a 2D grid. The Queen has finally approved a project to dig a straight canal from the top of the city to the bottom, connecting the royal reservoir to the farmland.

Lloyd must now choose a single column in the 2D city grid and demolish as few buildings (#) as possible in that column to create a clear vertical canal from the top to the bottom.

You must help Lloyd determine the minimum number of demolitions required to make any one column fully diggable (i.e., turn all cells in that column into .). He doesn't care which column — he'll just pick the cheapest.

Input

The first line contains two integers \(n\) and \(m\) \((1 ≤ n, m ≤ 100)\) — the number of rows and columns.

The next \( n \) lines, each with a string of \( m \) characters, represent the city grid:
. = empty land (no demolition needed)
# = building (needs to be demolished)

Output

Print a single integer — the minimum number of buildings that must be demolished in a single column to build the vertical canal.

Example

Input

4 5
.....
..#..
.....
..#..

Output

0

Input

4 5
#....
.##..
..###
..#..

Output

1

F. I Wanna Be You, Just For a Day

Time limit per test: 2 seconds
Memory limit per test: 1024 megabytes

After a wild body-switching incident, \(n\) people ended up in the wrong bodies. Initially, person \( i \) was in body \( i \), but now person \( i \) is in body \( p[i] \). The array \( p \) is a permutation of \( {1, 2, ..., n} \).

Medea needs to restore everyone to their original body using swap operations. In each swap, she can choose any two people and swap the bodies they're in.

Find the minimum number of swaps required to return everyone to their proper body.

Input

The first line contains a single integer \(n \) \( (1 ≤ n ≤ 1000)\) — the number of people.

The second line contains \(n\) space-separated integers \( p_1, p_2, …, p_n \) \((1 ≤ p_i ≤ n) \), where \( p_i \) is the current body of person \(i\).

Output

Print a single integer — the minimum number of swap operations required to restore everyone to their original body.

Example

Input

4
2 1 4 3

Output

2

Explanation

Swap (1, 2) and (3, 4). Total swaps = 2.

G. Fake Saint of the Year

Time limit per test: 2 seconds
Memory limit per test: 1024 megabytes

Elrise, fake saint and con-artist extraordinaire, has made it to the finale of the Grand Mages' Tournament. In the last trial, she must face off against multiple battle challenges, one after another.

In each challenge, she is given a sequence of magical attacks represented by integers. She must respond with spells of equal or greater strength to survive.

Before each challenge, she may rearrange her spells in any order. But she must use each spell exactly once, in some order, to block the enemy attacks—also one by one.

Elrise will win the challenge only if she can block all enemy spells, such that:

Each of her spells is ≥ the enemy spell at that turn.
Spells cannot be reused.
She must commit to a fixed order before the fight starts.

Input

The first line contains an integer \( n \) \((1 ≤ n ≤ 10^5) \), the number of spells in the challenge.

The second line contains \( n \) integers \(e_1, e_2, ..., e_n\), where \(e_i\) is the power of the enemy spell.

The third line contains \( n \) integers \(f_1, f_2, ..., f_n\), where \(f_i\) is the power of Elrise's spell \(i\).

Output

Print Victory if Elrise can reorder her spells to block all enemy spells. Otherwise, print Exposed — her cover's blown and it's over for her.

Example

Input

5
2 4 3 5 1
3 5 2 6 4

Output

Victory

Input

3
5 6 7
5 5 5

Output

Exposed

Help: Standard Input/Output

In programming competitions, problems usually require you to read from standard input and write to standard output.

What is Standard Input?

This is the input your program reads from the console. You don't need to open any files or create a GUI. Just read using language-specific commands like cin (C++) or input() (Python).

What is Standard Output?

This is where your program writes its answers — again, through the console. Use commands like cout (C++) or print() (Python). Make sure the output matches the required format exactly.

Sample Problem:

Given two integers \( A \) and \( B \), print their sum.

Input

3 4

Output

7

Sample Solution Template

In C++

#include <iostream>
using namespace std;

int main() {
    int A, B;
    cin >> A >> B; // Read two integers
    cout << A + B << endl; // Print their sum
    return 0;
}

In Python

# Read input, split by space, convert to int
A, B = map(int, input().split())
print(A + B)
